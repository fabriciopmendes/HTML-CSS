Capturando Fontes

De Textos da Web

Podemos baixar pelo Google Web Store extensões para captura de fontes de sites. Esse metodo é importante para quando encontrarmos uma fonte interessante e não sabemos qual é o nome dela. Uma das extensões é o Fonts Ninja, nele você pode encontrar todas as configurações daquela fonte para utilizar no seu site.

De Imagens

Não é possível identificar fontes que não são textuais por meios covencionais, ja que são outro tipo de arquivo. Existem sites capazes de identificar textos em imagens, é um processo mais complexo e existe risco de não identificar corretamente, mas é uma opção de uso. Sites como: What Font is, Font Squirrel, My Fonts. Nesses sites você enviar a imagem e ele faz uma leitura atrás de textos, é possivel ajustar alguns pontos da analise para melhorar o resultado.
